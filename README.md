# BRANDO

A Movie searching web application build with REACT and TMDB API

Website: https://brando-movie.herokuapp.com/

## Server

## Install

```sh
$ cd server
$ npm i
```

## Run

Nodemon should be pre installed to run this command

Create .env file

```text

NODE_ENV=
URI=
SECRET=
REACT_APP_MOVIE_API=
REACT_APP_BACKEND_API=
REACT_APP_API_KEY=
FRONT_END_URL=

```

```sh
$ cd server
$ npm start
```
