# BRANDO

A Movie searching web application build with REACT and TMDB API

Website: https://brando-movie.herokuapp.com/

## Server

## Install

```sh
$ npm i
```

## Run

Nodemon should be pre installed to run this command

Create .env file

```text

URI=
SECRET=
REACT_APP_MOVIE_API= 
REACT_APP_API_KEY=

```

```sh
$ npm start
```
